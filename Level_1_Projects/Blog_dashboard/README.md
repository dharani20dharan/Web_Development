# React Blog Dashboard

A simple and responsive blog dashboard built with **React** and **React Bootstrap**.  
This application demonstrates component-based design, props usage, and Bootstrap integration in React.

## Features
- Responsive navigation bar with links to **Home** and **Posts**
- Dynamic grid of blog post cards using React Bootstrap's **Card**, **Row**, and **Col**
- Each post displays a title, short description, and a **Read More** button
- Modular structure with reusable functional components

## Technologies Used
- React
- React Bootstrap
- React Router DOM
- Vite (for fast development setup)
