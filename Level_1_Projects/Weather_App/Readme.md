# Weather Dashboard

A modular weather dashboard that fetches and displays current weather data for any city using the OpenWeatherMap API. Built with HTML, CSS, and JavaScript following modern best practices.

## Features
- Real-time weather updates
- Search by city name
- Displays temperature, humidity, wind speed, and description
- Clean, responsive UI with weather icons
- Error handling for invalid cities and network failures

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+, async/await)
- OpenWeatherMap API
