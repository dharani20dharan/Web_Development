# Countdown Timer App

A simple and responsive countdown timer built using **React**, allowing users to set a custom duration, start, pause, and reset the timer. Built with functional components, props, useState, and useEffect.

## Features
- Set custom time in minutes and seconds
- Start, pause, and reset controls
- Real-time countdown display
- Auto-stop at zero
- Responsive and minimal UI

## Components
- **TimeSetter** – Input for minutes and seconds  
- **TimerDisplay** – Displays remaining time  
- **ControlButtons** – Handles start, pause, and reset  

## Tech Stack
React | HTML | CSS | JavaScript (ES6+)
