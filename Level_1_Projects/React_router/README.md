# React Router CV, Thermostat & Timer App

This project demonstrates a multi-page **React application** built with **React Router**, featuring personal CV sections and two interactive components — a **Thermostat** and a **Countdown Timer**.

---

##  Features

- **Multi-Page Navigation** using React Router  
- **CV Pages:** Home, Education, Skills, Projects, Experience, Achievements  
- **Interactive Components:**  
  - **Thermostat:** Temperature control simulation  
  - **Timer:** Countdown timer with start, pause, and reset  
- **Responsive UI** with custom CSS  
- **Functional Components & Hooks** (`useState`, `useEffect`)

---

##  Tech Stack

- **Frontend:** React (Vite)  
- **Routing:** React Router DOM  
- **Styling:** CSS  
- **State Management:** React Hooks
