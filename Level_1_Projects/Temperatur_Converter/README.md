# Temperature Converter Web App

A simple and responsive React application that converts temperatures between Celsius and Fahrenheit. Users can increase or decrease the temperature and toggle units with real-time conversion.

## Features
- Convert between Celsius (°C) and Fahrenheit (°F)
- Increment and decrement temperature values
- Instant conversion on unit toggle
- Responsive and minimal UI
- Built using functional components, props, and useState

## Tech Stack
- React (Vite)
- HTML5, CSS3
- JavaScript (ES6+)
