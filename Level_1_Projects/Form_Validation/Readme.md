# Community Event Registration Form

A user-friendly and responsive registration form built using HTML, CSS, and JavaScript with real-time input validation.

## Features
- Real-time form validation using JavaScript
- Clear error feedback and red borders for invalid fields
- Validations for name, email, phone, age, gender, and terms
- Modular, readable, and reusable code

## Validation Rules
| Field | Rule |
|--------|------|
| Full Name | Only letters and spaces, minimum 3 characters |
| Email | Must follow standard email format |
| Phone Number | Must contain exactly 10 digits |
| Age | Must be 18 or above |
| Gender | Must select one option |
| Terms | Must be accepted before submission |

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)

## Testing
Validated for:
- Missing fields  
- Invalid input formats  
- Edge cases like age 17 vs 18
