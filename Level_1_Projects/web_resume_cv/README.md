# Personal Resume Website

This project is a responsive HTML and CSS-based resume webpage designed for showcasing a fresher profile in Full-Stack Development and Artificial Intelligence. It highlights academic background, experience, and projects in a simple, professional layout suitable for IT industry portfolios.

## Features
- Fully responsive HTML and CSS design
- Clean and minimal layout
- Structured sections for profile, education, experience, projects, and certificates
- Easy to maintain and host on GitHub Pages or Netlify

## Tech Stack
- HTML5  
- CSS3  
- Git & GitHub for version control  
- Hosted on: Vercel
